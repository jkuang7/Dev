# Career Folder Intake Guide

Use this folder as the canonical input source for `/career`.

Recommended files:
- `Job Posting.md` (one target posting at a time)
- `Interview_Profile.md` (your constraints, timeline, goals)
- `JianKuang_Resume.docx` (or resume PDF/MD)
- `Aspida_Notes.md` (optional raw notes)

Fast workflow:
1. Paste the target JD into `Job Posting.md`.
2. Update `Interview_Profile.md` before each interview cycle.
3. Keep role-specific notes in `*_Notes.md`.
4. Run `/career foundation` once, then `/career target <company>`.

If you are applying to multiple roles, keep one `Job Posting.md` at a time and archive older JDs into `archive/`.
